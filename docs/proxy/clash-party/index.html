导入订阅

启动 Clash Party 后：

  • 进入配置或订阅管理页面
  • 粘贴你的机场订阅链接
  • 确认导入

选择节点开启代理

订阅导入成功后：

  1. 进入 节点 / Proxies
  2. 选择需要使用的节点（如 🇭🇰 香港 / 🇸🇬 新加坡 / 🇺🇸 美国）
  3. 确认节点生效

模式选择

根据使用场景选择合适的模式：

  • 规则模式（Rule）：推荐日常使用
  • 全局模式（Global）：所有流量走代理
  • 直连模式（Direct）：不使用代理
