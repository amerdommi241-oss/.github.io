v2rayN 使用教程

本文是关于 v2rayN 使用教程 的详细图文设置教程，帮助您快速配置并连接网络。

v2rayN 使用教程

v2rayN 是一款运行在 Windows 系统下的代理客户端，基于 v2ray / Xray / sing-box 内核开发，功能强大、协议支持全面。

支持的代理协议包括：

  • VMess
  • VLESS
  • Trojan
  • Shadowsocks
  • Socks
  • Hysteria2
  • Tuic

通过本文（2026 最新版）可完成 软件下载 → 订阅导入 → 节点选择 → 系统代理 → 路由模式 的完整配置流程。

使用订阅分组导入（推荐）

点击软件主界面顶部 「订阅分组」 → 订阅分组设置。

v2rayN 订阅分组

在弹出的窗口中点击 添加。

v2rayN 添加订阅分组

填写以下内容：

  • 别名：自定义名称
  • 可选地址（URL）：粘贴订阅链接

点击 添加 → 确定 保存。

v2rayN 订阅分组设置

添加完成后，点击：

订阅分组 → 更新全部订阅（不通过代理）

v2rayN 更新订阅

从剪贴板导入节点

复制机场提供的节点链接（需完整复制）：

  • VMess：vmess://
  • VLESS：vless://
  • Shadowsocks：ss://
  • Socks：socks5://
  • Trojan：trojan://

点击 服务器 → 从剪贴板导入批量 URL 即可完成导入。

v2rayN 剪贴板导入

扫描屏幕二维码导入

打开节点二维码图片，点击：

服务器 → 扫描屏幕上的二维码

v2rayN 扫描二维码

手动添加节点（进阶）

一般机场用户无需手动添加，以下适合自建或特殊配置用户。

支持手动添加的节点类型：

  • VMess
  • VLESS（Xray）
  • Shadowsocks
  • Socks
  • Trojan
  • Hysteria2
  • Tuic

路径统一为：

服务器 → 添加对应协议服务器 → 填写参数 → 确定保存

基础使用说明

选择节点

在主界面节点列表中：

  • 双击左键 直接启用节点
  • 或 右键 → 设为活动服务器

v2rayN 选择节点

启用系统代理

在任务栏右下角系统托盘中：

右键 v2rayN 图标 → 自动配置系统代理

图标变为红色表示已启用。

v2rayN 系统代理

路由模式说明

v2rayN 支持三种常用路由模式：

  • 绕过大陆（Whitelist）：仅国外流量走代理（推荐）
  • 黑名单（Blacklist）：除指定域名外均走代理
  • 全局（Global）：所有流量走代理

一般用户选择 绕过大陆模式 即可。

v2rayN 路由模式

开机自动启动

进入：

设置 → 参数设置 → v2rayN 设置

勾选 开机自动启动，点击确认保存。

v2rayN 开机启动

在线更新

支持在线更新以下组件：

  • v2rayN 客户端
  • Xray Core
  • sing-box Core
  • Geo 文件

点击 检查更新 即可。
