什么是 Clash Meta for Android？

Clash Meta for Android（现多更名为 Mihomo for Android）是目前安卓系统上绝对的正统王者工具。它集成了功能极其强大的 Mihomo (Clash Meta) 内核，相比原版增加了对 VLESS (Reality)、Hysteria2 以及各种高级分流规则的支持。

支持协议包括：VLESS、VMess、Trojan、Shadowsocks、Hysteria2、TUIC 等。

📥 软件下载

  • GitHub Releases: Clash Meta for Android Releases
  • 下载 .apk 文件后直接安装即可。

⚙️ 添加订阅教程

获取订阅地址

登录机场官网后台，复制 订阅链接。

⚠️ 建议关闭浏览器自动翻译功能，避免订阅链接被破坏。

新增配置

打开 Clash Meta：

  • 点击底部 配置
  • 点击右上角 ➕
  • 选择 URL 导入

填写订阅信息

  • 名称：随意填写
  • URL：粘贴机场订阅链接
  • 点击右上角 保存 并等待更新

订阅导入成功

看到配置文件出现在列表后，点击该配置使其生效（前面有蓝色勾选即为激活）。

🚀 启动代理连接

返回主页面开启代理

回到 主界面，将顶部开关打开，状态变为「运行中」即表示已成功连接。

🔄 节点切换方法

点击底部 代理 选项，进入节点列表，选择需要使用的节点（如 🇭🇰 香港 / 🇸🇬 新加坡 / 🇺🇸 美国）。

💡 路由模式说明

模式 说明
规则（推荐） 按规则分流，国内直连国外走代理
全局 所有流量走代理
直连 所有流量不走代理
