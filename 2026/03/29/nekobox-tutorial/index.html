什么是 Nekobox (猫盒)？

Nekobox，因其独特的猫耳图标被广大网友亲切地称为 “猫盒”。它是一款基于 sing-box 核心开发的开源代理客户端，最大的特色在于其几乎支持所有的现代代理协议，且由于其核心库非常先进，对于新协议的适配速度极快。对于那些追求”极致全能”的极客用户来说，猫盒是最佳选择。

支持协议包括：VMess、VLESS、Trojan、Shadowsocks、Hysteria2、TUIC、NaïveProxy、HTTP(S)、SOCKS5 等几乎所有主流协议。

📥 软件下载

  • GitHub Releases: Nekobox for Android Releases
  • 下载 .apk 文件后直接安装。

⚙️ 添加订阅教程

打开配置页

打开 Nekobox，点击底部 配置 选项卡，点击右上角 ✚ 号。

选择「从剪贴板导入」或「URL 导入」

提前复制好机场订阅链接，选择 从剪贴板导入 或 以 URL 添加配置文件。

填写订阅信息

填写名称与订阅 URL 后保存，等待订阅更新完成。

🚀 启动代理

订阅导入成功后，返回主界面，点击大的 启动按钮 开启代理。

首次启动会弹出 VPN 权限申请，点击 确认 授权。

🔄 节点切换

在主界面点击当前的节点名称，可以进入节点列表进行切换，选择延迟最低的节点。

💡 路由模式说明

模式 说明
绕过大陆（推荐） 国内直连，国外走代理
全局 所有流量走代理
直连 所有流量不走代理
