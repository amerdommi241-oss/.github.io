Midjourney 使用指南【2026年更新】：国内访问 + 共享账号 + 平替方案全攻略

Midjourney（MJ）是目前世界上最主流的 AI 图像生成工具，以画质精美、风格丰富、操作便捷著称，被设计师、插画师、广告人和创意工作者广泛使用。本文带你快速搞定访问方式和上手技巧。

Midjourney 是什么？

Midjourney 是一款以 Discord 为载体的 AI 绘图服务，你只需在 Discord 中向 MJ Bot 发送一段描述性文字（Prompt 提示词），它就能在约 1 分钟内生成 4 张高质量图像。

🎨 Midjourney 的核心优势

  • 画质顶级：生成的图像质感、光影、细节远超其他同类工具
  • 风格多元：从写实摄影到油画、插画、赛博朋克、吉卜力等风格一网打尽
  • 操作简单：通过自然语言描述即可生成，无需绘画基础
  • 版本迭代快：从 V5 到 V7 持续进化，细节控制越来越精准

国内如何访问 Midjourney？

Midjourney 依托 Discord 运行，而 Discord 和 Midjourney 官网在国内均需翻墙访问。

方案一：翻墙 + Discord 使用官方版（体验最完整）

步骤：

  1. 选择一个支持解锁 Discord 的优质机场（大多数主流机场均支持）
  2. 访问 Discord 官网 注册账号
  3. 访问 Midjourney 官网 订阅或加入免费服务器
  4. 在 Discord 中找到 MJ Bot，发送 /imagine 命令开始创作

💡 提示：Midjourney 已开放网页版（midjourney.com/imagine），登录后可直接在网页中创作，不再强制依赖 Discord。

高效提示词技巧

  1. 用英文描述：MJ 对英文 Prompt 理解更准确，可用 ChatGPT 或 Claude 辅助翻译
  2. 分层描述：主体 → 环境 → 光线 → 风格，层次越清晰效果越好
  3. 参考艺术家风格：加入 in the style of Studio Ghibli / by Greg Rutkowski 等关键词
  4. **善用权重 ::**：cat::2 dog::1` 让猫的比重是狗的 2 倍
