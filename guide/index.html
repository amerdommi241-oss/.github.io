科学上网软件使用指南

本指南汇总了目前主流的翻墙与科学上网客户端使用配置教程。覆盖 Windows、Android、macOS 与 iOS 全平台，帮助您快速上手，稳定访问海外网络服务。无论您是新手还是高阶玩家，都能在这里找到适合您的客户端工具。

Windows 客户端

Windows 客户端
微软 Windows 平台推荐代理软件（支持 Win 10 / Win 11）
Clash Verge Rev
Clash Verge Rev 基于 Mihomo 内核，代理功能强大，界面简洁易用并且支持自定义主题，是一个功能全面并且操作方便的强大客户端。
v2rayN
基于 v2ray 内核开发，功能强大且支持多种代理协议，包括 VMess、VLESS、Trojan、Socks、Shadowsocks、Hysteria2 和 Tuic 等。
clash-party
基于 Mihomo 核心的新一代网络代理工具，界面简介清晰，功能强大。支持所有主要桌面操作系统，能广泛满足不同用户的需求。
FlClash
基于 ClashMeta(Mihomo) 核心的网络代理工具，它的界面基于 Material You 设计，简单易用，开源且无广告。

Android 客户端

Android 客户端
安卓手机及平板端常用代理软件
FlClash
基于 ClashMeta(Mihomo) 核心的网络代理工具，它的界面基于 Material You 设计，简单易用，开源且无广告。
Clash Meta For Android
基于 mihomo 内核开发。CMFA 支持多种流行的代理协议，包括 V2Ray、Trojan、SS(R) 和 Socks 等，可以满足安全隐私保护方面的需求。
NekoBox
基于 sing-box 开发的多平台通用代理工具，支持多种代理协议，提供全面的网络代理解决方案。

MacOS 客户端

MacOS 客户端
苹果 macOS 桌面端代理软件（支持 Intel / Apple Silicon）
Clash Verge Rev
Clash Verge Rev 基于 Mihomo 内核，代理功能强大，界面简洁易用并且支持自定义主题，是一个功能全面并且操作方便的强大客户端。
Clash Party
基于 Mihomo 核心的新一代网络代理工具，界面简介清晰，功能强大。支持所有主要桌面操作系统，能广泛满足不同用户的需求。

iOS 客户端

iOS 客户端
苹果 iPhone 手机及 iPad 网络切换代理工具
Clash Mi
基于强大 Mihomo 核心的、易于上手的代理客户端，特别为希望快速使用机场订阅、享受智能分流和隧道代理的用户设计。
Shadowrocket
Shadowrocket是 iOS 平台上较早出现的支持 SS/SSR/V2ray 协议的客户端，坊间称之为“小火箭”。
Quantumult X
Quantumult X 简称“圈X”，是一款功能强大的网络工具，目前支持的协议：SS/SSR、V2Ray、VLESS、Trojan、HTTP(S)。
Egern
Egern 是一款功能丰富且强大的网络工具，专门用于代理、拦截和修改网络流量。
